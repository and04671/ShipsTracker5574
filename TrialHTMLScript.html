About

This is the Li(f)eBoat Ship Tracker webpage, a project for GIS 5574. It is powered by a Python webscraper using BeautifulSoup and Pandas.MMSI codes are used to search for a particular ship.

Click on a ship icon to learn more about that ship and its position.
