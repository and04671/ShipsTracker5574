Live Ship Tracker

by Cole Anderson, University of Minnesota

Welcome to the ship tracker!

This tool is a semester project for GIS5574 at the University of Minnesota. It utilizes Python code to track the location of a handful of ships on the world map.

You can click each ship icon to learn more about the selected vessel. Ships are listed on the right with a link to further information about that vessel
